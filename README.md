# okinawa-lodging-tax

沖縄県宿泊税の税額算定・集計・帳簿保存・領収書発行・県提出用データ出力を行うSaaS。
将来的に複数宿泊施設（マルチテナント）への提供を前提とする。

## ドキュメント

- [要件定義（たたき台）](./docs/requirements.md)

## ステータス

要件定義フェーズ。リポジトリは要件整理用に立ち上げたばかり。

## 想定スコープ（初期フェーズ）

- 宿泊実績の登録（手入力 / CSV取込）
- 宿泊税の自動計算（税率切替・上限考慮）
- 日次・月次集計
- 領収書発行（宿泊料金と宿泊税を分離表示）
- 月次申告用データの出力
- 帳簿・証憑の保存と検索

## 非スコープ（初期フェーズでは対象外）

- PMS機能（予約・在庫・顧客管理）
- 会計仕訳・総勘定元帳
- eLTAX等オンライン申告API連携（将来拡張）
